## Development server

`ng serve` para arrancar servidor en `http://localhost:4200/`

## Build

`ng build` construir en `dist/`

## Running unit tests

`ng test`

## Running end-to-end tests

`ng e2e`

Se han generado los scripts para desplegarlo en heroku.

## Para desplegar en producción se ejecuta
`ng build --aot -prod`
`node server.js`
